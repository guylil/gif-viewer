# gif-viewer

A Simple Vuejs Gif Viewer

## Project setup

```
clone the repo

cd gif-viewer

npm install

npm run serve

Have Fun :)

```

### Don't forget to get your Giphy API Key
```
Go to https://developers.giphy.com/docs/api

To get your key (it's free :)
```


### Built with vuejs cli

### Compiles and minifies for production
```
npm run build
```
